# Favorite Beings Personal Webpage

## Description
This is a simple, responsive personal webpage showcasing two favorite beings â€” "Majito" and "Peloss". The page allows users to switch between the two using clickable cards. Each selection displays a carousel of images along with a descriptive text about the chosen subject.

The background color of the page can be changed randomly by clicking a button, adding a dynamic and playful touch.

## Features
- Responsive design using custom CSS and Google Fonts (Poppins).
- Dynamic background color changer.
- Image carousel with next/previous controls for each subject.
- Modular JavaScript managing UI updates and carousel behavior.
- Clean, accessible markup structured for easy extension.

## Technologies
- HTML5
- CSS3 (custom styling)
- JavaScript (vanilla)
- Google Fonts (Poppins)

## How to Run
1. Open `index.html` in any modern web browser.
2. Use the buttons and cards to interact with the page:
   - Click "Cambiar color de fondo" to change the background color.
   - Click "Mi Mojarrita" or "La Pelos" to view respective image carousels and descriptions.
   - Navigate carousel images with left/right arrows.

## Notes
- Images referenced in the carousel must be located in the same directory or properly linked.
- Designed to be simple and easily customizable.
- Ideal for personal portfolios or small presentations.
